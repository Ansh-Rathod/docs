Pimosa supports a wide range of file formats for video, audio, and image processing. Here's a detailed breakdown of the supported formats:

---

## Video Formats

Pimosa can process video files in the following formats:

- **MP4** (`.mp4`)
- **MKV** (`.mkv`)
- **AVI** (`.avi`)
- **MOV** (`.mov`)
- **FLV** (`.flv`)
- **WEBM** (`.webm`)
- **WMV** (`.wmv`)
- **MPG** (`.mpg`, `.mpeg`)
- **3GP** (`.3gp`)
---

## Image Formats

Pimosa allows you to work with the following image formats:

- **JPG** (`.jpg`, `.jpeg`)
- **PNG** (`.png`)
- **WEBP** (`.webp`)
- **BMP** (`.bmp`)
- **TIFF** (`.tiff`)
- **HEIC** (`.heic`)
- **GIF** (`.gif`)




---

## Audio Formats

The app supports a variety of audio files, including:

- **MP3** (`.mp3`)
- **AAC** (`.aac`)
- **OGG** (`.ogg`)
- **WMA** (`.wma`)
- **OPUS** (`.opus`)
- **WAV** (`.wav`)
- **FLAC** (`.flac`)
- **ALAC** (`.alac`)
- **APE** (`.ape`)
- **AIFF** (`.aiff`)
- **DSF** (`.dsf`)
- **DFF** (`.dff`)
- **M4A** (`.m4a`)
- **MKA** (`.mka`)
- **MIDI** (`.midi`, `.mid`)
- **RA** (`.ra`)
- **RAM** (`.ram`)



---

## Additional Notes

- If you need support for a specific file format not listed here, please contact us at **pimsoa.app/support**.

Pimosa is constantly evolving to support more formats and provide a better user experience.
